# Requirements

  - Python 2.7, Python 3+
  - `pyevolve` python library
  - `ACO-pants` python library

# Installation

```sh
$ sudo apt-get install python-pyevolve
$ sudo apt-get install python-matplotlib
$ pip3 install ACO-Pants
```
